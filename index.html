Conhecimentos

HTML

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade.

CSS

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets, onde se define estilos para seu projeto web.Com efeitos de transição, imagem,imagem de fundo/Background e outros, que dão um estilo novo e elegante a seus projetos web. Ou em todos os aspectos de design do layout da página.

Manutenção de PCs

Fazemos diversos serviços de manutenção de computadores, como limpeza, troca e instalação de componentes, instalação de programas e apps, formatação, implementação de redes, remoção de virus, entre outros, bastando nos contactar através da sessão contato deste site, ou em nossas redes sociais.

Javascript

Javascript é uma linguagem de programação de alta complexidade, mas de fácil uso, voltada para criar elementos em aplicações web, como sites, aplicativos e sistemas online. O mercado tem cada vez mais se direcionado para web, o que ressalta o quanto a JavaScript ganhou papel de protagonismo.

Java

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems. Em 2008 o Java foi adquirido pela empresa Oracle Corporation.

Git e GitHub

Git é um sistema de controle de versão desenvolvido por Linus Torvalds (o criador do Linux).GitHub, a maior rede social para desenvolvedores do mundo, é um serviço baseado em nuvem que hospeda o Git. Permitindo a desenvolvedores colaborarem fazendo mudanças em projetos compartilhados enquanto mantêm um registro detalhado do seu progresso.
